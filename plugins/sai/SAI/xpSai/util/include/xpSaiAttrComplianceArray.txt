/*
 * Each SAI object has some number of attributes. Each attribute has some
 * properties (MANDATORY_ON_CREATE, CREATE_ONLY, CREATE_AND_SET) that has to be
 * Generated array will show attribute compliance report for each attribtute
 * property. Arrays are defined in xpSaiAttrComplianceArray.txt file.
 *
 * The meaning of the array is:
 *    id;                -- sai_attr_id_t -- attribute id
 *    xpSaiSupport;      -- bool          -- is attribute supported by SAI Adapter?
 *    validForCreate;    -- bool          -- is attribute valid for create?
 *    validForSet;       -- bool          -- is attribute valid for set?
 *    validForGet;       -- bool          -- is attribute valid for get?
 * Please note, some of the attributes are not supported. List of such attributes
 * is located in xpSaiNotSupported.txt file.
 */
